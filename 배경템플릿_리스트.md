# 니냥내냥 — 배경 템플릿 제작 리스트

> **배경(씬) 아트만** 만들면 돼요. 고양이·이름·뱃지·별점·프레임·일련번호 등은 전부 **코드가 위에 얹어요.**

## 📐 공통 스펙
- **비율**: 3:4 세로 (카드 모양)
- **해상도**: 권장 **1500 × 2000 px** 이상 (실물 인쇄 대비 클수록 좋음)
- **포맷**: PNG 또는 JPG (투명 필요 없음, 꽉 찬 배경)
- **스타일**: **단순화 팝아트** — 색면 + 망점(Ben-Day) + 선버스트. 장면은 알아보되 디테일 과하지 않게
- **안전영역**: **가운데~아래는 비교적 단순하게** (여기에 고양이가 앉아서 가려짐 / 복잡하면 고양이가 안 떠 보임)
- ❌ **넣지 말 것**: 고양이, 글자, 뱃지, 별점, 아이콘, 금테 프레임, 일련번호 — 전부 코드 오버레이
- **파일명 = 아래 key** (예: `sunset.png`) 로 주면 코드에 바로 매핑돼요
- 참고: 부족(털색)별 색변형은 코드에서 **가벼운 색틴트**로 처리 → 씬당 **1장만** 만들면 됨

---

## ① 일반 씬 (14개) — 캐치 맥락으로 자동 배정

| key | 이름 | 설명 | 배정 |
|-----|------|------|------|
| `day` | 낮·동네 골목 | 밝은 한낮, 담벼락/골목, 뭉게구름 | 낮 |
| `sunset` | 노을 | 주황·핑크 노을, 도시 실루엣 | 노을 시간대 |
| `night` | 밤하늘 | 별·달, 짙은 남보라 | 밤 |
| `dawn` | 새벽 | 푸른빛, 가로등 잔광 | 새벽 |
| `rain` | 비 오는 날 | 빗줄기, 물웅덩이 반사 | 비 |
| `snow` | 눈 오는 날 | 함박눈, 차가운 톤 | 눈 |
| `fog` | 안개 | 뿌연 회색, 몽환 | 안개 |
| `spring` | 벚꽃 봄 | 벚꽃 흩날림, 분홍 | 봄 |
| `summer` | 여름 바다 | 파란 바다·하늘, 수박빛 | 여름 |
| `autumn` | 가을 단풍 | 단풍·은행, 주황노랑 | 가을 |
| `neon` | 네온 시티 | 밤 네온사인, 시티팝 | 밤(도시) |
| `rooftop` | 옥상 | 옥상에서 본 동네 전경 | 장소 |
| `market` | 시장 골목 | 재래시장·포장마차 불빛 | 장소 |
| `river` | 한강/강변 | 강변, 다리, 노을 반사 | 장소 |

## ② 레전드 전용 특수 씬 (5개) — 호화롭게

| key | 이름 | 설명 |
|-----|------|------|
| `lg_aurora` | 오로라 | 초록·보라 오로라 커튼, 신비 |
| `lg_petal` | 꽃비 들판 | 꽃잎 폭우, 몽환 들판 |
| `lg_starsea` | 별바다 | 은하수·쏟아지는 별 |
| `lg_torii` | 신사 토리이 | 붉은 토리이, 영험한 빛 |
| `lg_rainbow` | 무지개 구름 | 무지개·반짝, 환상 (※ 네 샘플 그 느낌) |

> 레전드 씬은 **잘 안 나오니까** 한 장 한 장 화려하게 — 뜰 때마다 "와!" 나오게.
