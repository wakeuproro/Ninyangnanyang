> **1순위: 니냥내냥** (니냥=동네/길냥이 발견·수집, 내냥=내 반려묘 카드제작 → 2-트랙 컨셉을 이름에 그대로 담음. 발음 찰짐, 캣맘 감성. 단 전 동물 확장 시 "냥"이 고양이에 잠기는 점만 추후 고려)

## 0. 한 줄 정의

> **실제 동물을 사진으로 "잡아" 나만의 수집 카드로 만들고, 마음에 드는 카드는 실물로 제작·배송받는 현실 연동형 동물 도감.**

영감: 인스타 @promppy_com 에서 본 외국 앱 **CatchCat** — 카메라로 실제 고양이를 스캔(머리/몸통 인식) → 캔으로 유인 → 캐치 → AI가 TCG 카드(코스트/파워/부족/능력 + 희귀도 + 일련번호)로 변환 → 도감 수집.

**우리의 차별점 (원조는 디지털 수집에서 끝):**
1. **디지털 → 실물 카드 비즈니스** (한국 포토카드·굿즈 문화 활용)
2. **한국형 정서·커뮤니티** (길냥이·캣맘·반려동물 추모)
3. **전 동물 확장 구조** (고양이로 시작, 강아지·전체 펫으로 확장)

## 1. 핵심 전략 결정 (확정)

### 🎯 스코프: 고양이로 시작 → 전 동물 확장
- **이유**: 좁게 시작해 열성 니치(캣맘·길냥이 커뮤니티)를 확실히 이긴 뒤 확장. 길고양이는 "우연히 발견해 캐치"하는 게임 메커니즘과 가장 잘 맞음(강아지는 목줄·주인 동반이라 야생성↓).
- **단, 첫날부터 데이터는 종 무관 설계**: `animal_type` 필드로 고양이→강아지→전체 펫 확장이 "코드 재작성"이 아니라 "설정 추가"가 되게.

### 🔀 자연스러운 2-트랙 구조
| 트랙 | 대상 | 역할 |
|------|------|------|
| **발견/수집 게임** | 🐱 길고양이 | 밖에서 우연히 캐치, 도감 채우기 → 재미·바이럴 |
| **카드 제작 상품** | 🐶 내 반려동물(전 동물) | 내 동물을 전설 카드로 + 추모 카드 → 매출 |

→ 종을 늘려도 컨셉이 안 흐려짐. 고양이는 게임 엔진, 내 펫은 매출 엔진.

## 2. 타겟 사용자

| 층 | 누구 | 핵심 욕구 | 지불 의향 |
|----|------|----------|----------|
| **수집러** | 10~30대, 포토카드·굿즈 수집족 | 수집욕·도감 컴플리트·희귀템 | 중 (실물카드·가챠) |
| **반려동물 집사** | 고양이·강아지 키우는 사람 | **내 동물을 전설 카드로** / 추모 | **매우 높음** |
| **길냥이 커뮤니티** | 캣맘·동네 동물 관찰러 | 동네 동물 기록·공유·보호 | 낮~중 (명분 소비) |

> 💡 **핵심 인사이트**: 원조는 "길에서 만난 고양이"에 집중하지만, **돈은 "내 동물"에서 나온다.** 자기 반려동물을 전설급 카드로 만들어 실물로 받는 것 — 지불 의향 압도적. 특히 **무지개다리 건넌 반려동물 추모 카드**는 가격 저항이 거의 없는 감성 시장.

## 3. 핵심 게임 루프

```
탐색(카메라 스캔) → 츄르(미끼)로 유인 → 캐치(머리/몸통 인식)
   → AI가 카드 생성(희귀도 랜덤) → 도감 등록(#일련번호)
   → XP·코인 획득 → 일일 미션/미니게임 → 실물 주문
```

- **일일 캐치 제한** (예: 무료 3회/일) → 재방문 + 가챠 욕구
- **희귀도 랜덤** (Common → Uncommon → Rare → Legendary) → "한 번 더" 도파민
- **츄르(미끼) = 소비 재화** → 가챠/광고 시청/구매로 충전. 일반→프리미엄 츄르(희귀냥 등장 확률↑)로 등급화 가능
- 카드 구성(원조 참고): 코스트·파워 / 부족(Tribe) / 키워드(Guard 등) / 능력(ability) / 희귀도 / 일련번호(#000014)

## 4. 수익화 모델 (★ 핵심)

수익을 4개 엔진으로 분리해 겹친다. **게임은 무료로 풀고, 돈은 옆에서.**

### 엔진 A — 실물 카드 (진짜 매출 · 현실 연동) ← 1순위
- POD(주문 제작): 디지털 카드 중 골라 **실물 트레이딩 카드 인쇄·배송**
- 가격감(예시):
  - 단품 카드 3,500~5,000원
  - 추모·반려동물 프리미엄 카드 8,000~15,000원
  - **도감 N장 채우면 풀 앨범 굿즈** 2~3만원 (AOV↑, 배송비 흡수)
- 원가: 카드 인쇄 장당 수백 원대 → 마진 확보. 케이팝 포카가 장당 2~5천원에 거래되는 문화 → 가격 저항 낮음
- 운영: 초기엔 POD 외주 직접 → 주문 쌓이면 자동화

### 엔진 B — 구독 (반복 매출)
- **프리미엄 패스** (월 3,900~5,900원): 일일 캐치 무제한 · 희귀도 부스트 · 독점 프레임/시즌 한정 테마 · 광고 제거 · 실물 주문 할인
- 타겟: 집사층·헤비 수집러

### 엔진 C — 츄르/코인 가챠 + 광고 (소액결제·체류)
- **츄르(미끼) 충전**: 소액 IAP 또는 **리워드 광고 시청 = 츄르 1개**
- 코인으로 카드 프레임·배경·시즌 이벤트 참가

#### 먹이(미끼) 5단계 — "비싼 먹이 = 예쁜 카드 확률↑" (기획 확정 6/25)
> 핵심: 먹이 등급 = **희귀도 보너스**. 규칙엔진의 기존 보너스(밤/눈/최초발견)와 동일 메커니즘 → deterministic·explainable 유지. 비쌀수록 레전드 임계값 넘기 쉬움.

| 단계 | 먹이 | 레전드 보너스(점수) | 가격(츄르) |
|---|---|---:|---:|
| 1 | 🐟 츄르(기본) | +0 | 무료 |
| 2 | 🥫 통살 캔 | +2 | 5 |
| 3 | 🍗 황태 트릿 | +4 | 15 |
| 4 | 🍣 참치 회 | +7 | 40 |
| 5 | 👑 황금 만찬 | +11 | 100 |

- 최고급도 **레전드 보장은 아님**(좋은 스탯+맥락+먹이 겹쳐야) → 희소성 유지. 단 확률은 확실히↑.
- 사진마다 시드가 달라 → 여러 번 잡으면 **비싼 먹이일수록 레전드가 더 자주** = 체감 확률.
- **경제 루프**: 츄르 재화 ← 무료(일일·출석)/광고/💰IAP → 먹이 구매 → 던지기 → (비쌀수록) 예쁜 카드.
- ✅ **항상 카드는 나옴**(돈 잃는 도박 아님, 예쁨 확률만 삼) = 건전·거부감↓.
- ⚠️ **규제 플래그**: 유료로 확률 부스트 판매 = 한국 **확률형 아이템 확률 공시 의무**(2024 게임산업법) 대상 → 정식 출시 시 확률 공개 필요(파일럿/MVP는 OK).
- **구현 2슬라이스**: A) 먹이 선택 UI + 엔진 보너스 연결(통화 차감 X, 데모) → B) 츄르 재화·일일제한·구매·광고(경제).

### 엔진 D — 제휴·명분 (브랜드 + 부가)
- **지역 제휴**: 고양이 카페·펫샵을 "스폰서 핫스팟"으로 → 한정 카드, 방문 유도
- **동물보호 연계 (길냥이 후원) ★**: 수익 일부를 길냥이·유기동물에 환원. "고양이로 장사한다" 비판을 명분으로 전환 + 캣맘 커뮤니티 자발적 바이럴.
  - **투명하게 노출 (그린워싱 의심 방지)**: 결제 시 기부분 명시(예: 실물 카드 1장당 OOO원 후원), 앱 내 **누적 후원 카운터**("모은 후원금 ₩XXX · TNR OO마리")
  - **후원 뱃지·한정 홀로 카드**: 후원 참여 집사에게 특별 뱃지/한정 카드 → 명분 + 수집욕 동시 자극
  - **공식 제휴처 명시**: 동물권 단체·지역 보호소와 협약 (신뢰 핵심)
  - **사용처(구체적일수록 신뢰↑)**: TNR 중성화 / 겨울 급식소·길냥이 집 / 유기묘 보호소 / 구조 치료비
  - **사업적 이득**: 캣맘 바이럴 + 언론/SNS 스토리("게임으로 길냥이 돕는 앱") + 사회적가치 = 정부지원·여성창업 지원금 가산점

> **우선순위**: ① 무료 캐치로 수집 재미 검증 → ② **실물 카드(A)를 첫 매출원** → ③ 구독/가챠는 유저 쌓인 뒤. 실물 카드가 "가상↔현실" 컨셉의 핵심이자 가장 설명 없이 팔리는 상품.

## 5. 미니게임 / 체류시간 전략

수집만으론 매일 안 들어옴. 재방문·체류 장치:
- **카드 배틀**: 내 동물 덱 vs 친구/AI — 가벼운 하스스톤식 (원조의 Battle 스탯 활용)
- **일일 출석 · 동네 랭킹**: "오늘 우리 동네에서 가장 많이 발견된 동물"
- **도감 컴플리트 보상**: 특정 세트 완성 시 한정 카드 → 실물 앨범 주문 트리거
- **시즌 이벤트**: 핼러윈/설날 한정 스킨 → 한정 카드 FOMO

### 냥체인지 (친구 교환 · 소셜 바이럴) ★
> "거래(현금)"가 아니라 "친구랑 바꿔보는 재미"로 포지셔닝. 따뜻함 유지 + 바이럴 엔진.
- **카드↔카드만, 현금 0** (안전선 — 현금 거래 붙이면 사행성·게임법 리스크 + 감성 붕괴)
- **중복 카드 교환**: 같은 동네 냥이 중복분을 친구 거랑 스왑 (포켓몬 카드 문화)
- **양방 수락 스왑**: 친구 도감 구경 → "이 냥 갖고싶다" → 내 카드 제안 → 둘 다 수락 시 성사
- **지역 한정 카드**: "강남 냥이"는 그 동네 친구한테만 → 도감 100% = 다른 동네 친구와 교환 필수 → **친구 초대 동기(바이럴)**
- **선물하기**: 교환 없이 일방 선물도 가능 (더 훈훈)
- **냥체인지 데이**: 주말 한정 교환 이벤트 + 희귀냥 등장
- **수익 연결(은근하게)**: 교환=무료 → 친구 유입 → 실물카드/구독 전환. 프리미엄은 교환 슬롯 확장·랜덤(다른 동네) 매칭 정도만.

## 5.5 카드 생성 파이프라인 (★ 핵심 기술 결정 — 저비용 구조)

> 핵심: **카드를 통째로 생성형 AI로 만들지 않는다.** 디자인 템플릿을 미리 만들고, 찍은 고양이는 누끼(배경제거)만 따서 등급에 맞는 템플릿에 합성. 셀러비서Pro "AI 비용 1/10" 원칙과 동일.

```
사진 촬영
 → ① 누끼(배경 제거)          ← 생성 아님, 거의 무료
 → ② 품종·털색 분류 → 타입      ← 비전 분류 (치즈냥/고등어…)
 → ③ 규칙엔진 → 스탯·희귀도      ← LLM 아님, 무료·재현 가능
 → ④ 희귀도에 맞는 템플릿 선택   ← 미리 디자인한 SVG/PNG 프레임
 → ⑤ 캔버스 합성 (누끼+프레임+스탯) ← 클라이언트, 무료
 → 카드 완성
```

**누끼 방법(거의 공짜):** 온디바이스 ML(TensorFlow.js/MediaPipe) / iOS Vision 피사체 분리 / 정 안 되면 저가 누끼 API.

**아트 방향 확정 (PoC 검증 완료 2026-06-24): 실사 누끼로 간다. 일러스트화는 폐기.**
- 실제 길냥이 사진으로 누끼+일러스트 두 버전을 직접 만들어 비교한 결과, **일러스트화는 "내가 진짜 만난 그 고양이"라는 정체성·소장가치를 깎음.** 펫덱스의 가치는 예쁜 그림이 아니라 **authenticity**(실존하는 그 냥이) — 케이팝 포카가 "실제 그 사람"이라 가치 있는 것과 동일.
- 따라서 모든 등급이 **실사 누끼 기반**. 등급 차별화는 일러스트가 아니라 **포일/프레임 마감**으로만 줌(Common 무광 ~ Legendary 풀 홀로그램). → 생성형 비용 사실상 0 유지.
- 사진 퀄리티는 **가이드 캡처(아래)**로 통제. 못 찍으면 일러스트로 때우지 말고 **재촬영** 유도.

**등급별 비용 차등(모두 실사):**
| 등급 | 처리 | 생성형 비용 |
|------|------|------------|
| Common~Rare (90%+) | 누끼 + 템플릿 합성 + 무광/약포일 | 0 |
| Epic | 합성 + 강한 포일 프레임 | 0 |
| Legendary (희귀) | 합성 + 풀 홀로그램·박 마감 | 0 |

- 이름·한 줄 대사(LLM 텍스트)는 토큰 적고 배치·캐시 가능 → 부담 X
- 템플릿은 미리 제작 → 디자인 강점(김지선) 활용 포인트
- 실물 인쇄도 누끼+템플릿 합성본을 그대로 고해상도 출력 → 일관성 확보

### 카드 디자인 방향 (확정): 홀로그램/포일 — 케이팝 포카 감성 "소장각"
> 원조의 꽃밭 그림체(촌스러움)는 버림. 세련된 포일 카드로 간다.

- **📱 디지털**: 폰 자이로/마우스 각도에 따라 포일 무지개가 흐르는 효과 (CSS 홀로그램 카드, 오픈소스 검증된 기법). 생성형 0원.
- **희귀도 = 반짝임 강도로 차등**: Common 무광 → Rare 은은한 포일 → **Legendary 풀 홀로그램+글레어**. (레전드 특별함을 "생성형 그림"이 아니라 "포일 마감"으로 줌 → 비용↓)
- **🖨️ 실물**: 홀로그램 PET 카드 / 박(foil stamping) / 스팟 UV. 등급↑ = 마감 업그레이드 → 레전드 실물이 더 비싸도 납득되는 자연스러운 가격 차등.

### 배경 아트 스타일 (확정): 맥락 살린 "단순화 팝아트"
> 추상 팝아트(맥락 없음) ❌, 지브리(너무 디테일) ❌ → **그 중간**: 캐치 맥락(시간대·장소·날씨·계절)을 알아볼 수 있게 살리되, 장면을 **팝아트풍으로 플랫·볼드하게 단순화**.
- 배경이 단순해야 위에 얹은 **실사 누끼 고양이가 주인공으로 떠 보임** (PoC로 확인 — 디테일 배경은 시선 분산).
- 맥락 반영: "노을에 잡으면 노을 배경, 눈올 때 설경" → 단, 노을/골목/설경을 팝아트 색면+망점+선버스트로 단순화한 버전.
- 배경은 **코드(SVG/CSS) 템플릿**으로 제작 → 생성형 비용 0, 등급/타입/맥락별 색·요소 변형으로 무한 재활용. (실물 인쇄도 벡터라 쨍함)
- ※ 템플릿 실제 제작은 **추후 코딩 단계에서** (지금은 느낌만 검증 완료, 만들지 않음).

### 배경 템플릿 구성 (기획 확정 6/25 — 제작은 추후, 클로드디자인 활용 예정)
> 핵심: 배경은 **2축 곱하기 구조**. 씬 갯수만 늘리지 않고 곱셈으로 체감 다양성 폭증 → 적은 제작으로 도감이 안 지루함.

```
도감 체감 다양성 = [맥락 씬] × [타입(털색) 색변형] × [희귀도 마감]
```
예: 같은 "노을 씬"도 치즈냥=주황노을 / 고등어냥=블루노을 / 까만냥=핑크노을 (+레어면 포일) → 씬 1개가 수십 가지로 보임.

**① 맥락 씬 ~14종** (캐치 맥락으로 자동 배정 · CSS팝아트라 색변형은 코드로 공짜)
- 시간: 낮·골목 / 노을 / 밤하늘 / 새벽
- 날씨: 비 / 눈 / 안개
- 계절: 벚꽃봄 / 여름바다 / 가을단풍
- 장소: 네온시티 / 옥상 / 시장골목 / 한강

**② 레전드 전용 특수 씬 5종** (잘 안 나오니 풍성하게 — 뜰 때마다 잭팟 느낌)
- 오로라 / 꽃비 들판 / 별바다 / 신사 토리이 / 무지개 구름

**③ 희귀도 마감 5단계** (씬 위에 얹는 코드 오버레이 — 각도 반짝은 이미지로 불가)
- Common 무광(망점만) → Uncommon 약포일 → Rare 포일+굵은테 → Epic 홀로+빛줄기 → Legendary 풀홀로+파티클(꽃잎/별)+금테

**제작 규모/순서**
- 디자인 부담 핵심 = 레전드 특수 5 + 맥락 씬 베이스. 타입 색변형·마감은 코드 곱셈.
- **MVP는 씬 5~6 + 마감 3**으로 시작 → 이후 점진 확장.
- **시즌 업데이트로 배경 추가**(핼러윈·크리스마스·설날…) = 재방문/재캐치 동기(FOMO) + 도감이 계속 자람.
- 일부 씬은 희귀 조건에서만 등장(눈 씬=눈올 때만 등) → 수집 성취감.

### 카드 앞/뒷면 분리 (확정): 풀아트 앞 + 정보 뒤
> 실제 TCG처럼 양면 구성. 앞은 소장·비주얼, 뒤는 데이터.
- **앞면(Front) — 풀블리드 비주얼**: 배경 + 고양이 이미지를 카드 전체로(full-bleed). 올리는 정보는 **희귀도 뱃지 + 이름/별명** 정도만. 깔끔·소장각 극대화.
- **뒷면(Back) — 정보**: 스탯(예쁨·통통함·친화력·카리스마) · 타입/부족 · 능력 텍스트 · 캐치 기록(지역·시간·날씨·날짜) · 일련번호 · 친밀도 등 나머지 전부.
- 앱에선 카드 탭/플립으로 앞↔뒤 전환, 실물도 양면 인쇄.

### 가이드 캡처 (★ 촬영 = 게임 = 품질 통제)
> 실사로 가는 대신 사진 품질을 앱이 통제. 막 찍은 흐린 사진 방지 → 누끼 깔끔 → 카드 예쁨. 원조 CatchCat이 이미 쓰는 UX(Body found/Head found/3 tries/캔 드래그)를 차용.

- 🎯 **프레임 가이드**: 고양이를 가이드 네모 안에 맞춰야 캐치 가능 (머리+몸통 인식)
- 💡 **품질 게이트(캐치 전 자동 체크)**: 밝기(너무 어두움/역광) · 블러(흔들림) · 피사체 크기(너무 작음) · 단일 피사체 → 미달 시 *"조금만 더 밝은 곳에서 다시!"* 재촬영 유도
- 🐟 **재미 요소**: 츄르 드래그로 유인, 시도 횟수(3/3), 잘 찍으면 보너스 XP
- ✨ **품질 = 보상 연결**: 잘 찍을수록 누끼 깔끔 → 카드 더 예쁨 → "잘 찍자" 동기
- 일석삼조: ① 누끼 품질 확보 ② 캐치가 미니게임화(체류↑) ③ 저품질 사진 방지. (길냥이 사진은 배경이 단순해 가이드만 잡으면 누끼 잘 먹힘)

## 7. MVP 범위 (현실적 컷)

전부 만들면 못 끝남. **첫 버전은 "수집 루프"만**, 결제는 뒤로.

### MVP v0 — 수익화 전, 재미 검증
1. 사진 촬영/업로드 → AI가 카드 1장 생성 (희귀도 랜덤)
2. 내 도감(컬렉션) 그리드 + 카드 상세(스탯/일련번호)
3. 일일 캐치 제한 + XP/코인
4. 카드 공유(인스타) — **무료 바이럴 엔진**

→ "사람들이 모으고 자랑하나?" 검증되면 → **실물 주문 버튼** 붙여 첫 매출(엔진 A).

### 이후 단계
- v1: 실물 카드 주문(엔진 A) + 내 반려동물 카드
- v2: 구독(B) + 츄르 가챠/광고(C)
- v3: 카드 배틀·동네 랭킹 + 지역/보호 제휴(D)

## 8. 기술 스택 제안
