# 니냥내냥 — 카드 프레임 오버레이 제작 스펙

> 목업(AURORA / 노을이) 수준의 카드를 만들려면 **프레임(카드 테두리·장식)을 디자인 PNG로** 받아야 해요.
> 배경처럼, 프레임도 "디자인 자산 + 코드 텍스트" 구조.

## 🧱 카드 = 4겹 레이어
```
① 배경 PNG (이미 완성 — public/bg)
② 고양이 누끼 (코드 합성)
③ 프레임 오버레이 PNG (★ 이번에 만들 것 — 투명 가운데)
④ 코드 텍스트 (이름·일련번호·부족·별점 — 동적이라 코드가 얹음)
```

## 📐 프레임 오버레이 스펙
- **비율/크기**: 3:4, **1086 × 1448** (배경과 동일 캔버스 → 딱 겹침)
- **포맷**: **투명 PNG** (가운데는 뚫려서 배경+고양이가 보임)
- **그릴 것 = 카드 "테두리·장식"만**:
  - 바깥 금테/포일 테두리, 모서리 장식(보석·날개·필리그리 등)
  - 좌상단 **등급 뱃지 자리** (LEGENDARY 리본 등 — 등급 글자는 빼고 "빈 명패"로, 코드가 글자 채움)
  - 우상단 **부족 아이콘 자리** (빈 금색 동그라미 — 코드가 고양이 얼굴/털색 넣음)
  - 하단 **이름 자리**(가운데, 비워둠 — 코드가 홀로 이름) + **컬렉터/일련번호 명패**(좌/우)
- ❌ **넣지 말 것**: 고양이, 실제 이름 글자, 실제 일련번호, 배경 그림 (전부 ②③④에서 처리)
- **안전영역**: 가운데(고양이 자리)는 완전히 투명하게

## 🎴 몇 장 만들까 (최소부터)
부담 줄이려면 **2장**부터 시작:
1. **`frame_base.png`** — 일반 등급용 우아한 기본 프레임 (common~epic 공용, 등급 색은 코드가 뱃지로 표시)
2. **`frame_legend.png`** — 레전드 호화 프레임 (AURORA처럼 금테·보석·날개)

→ 잘 되면 등급별로 `frame_rare`, `frame_epic` 등 추가 (점진).

## 🔧 코드가 하는 일 (내가 연결)
- 프레임의 정해진 자리에 **이름(홀로)·일련번호·부족·별점**을 정확히 얹음
- 등급별로 뱃지 색/별 갯수 자동
- 고양이는 프레임 안쪽에 자연스럽게(접지 그림자 포함)

> 즉 **너는 "빈 액자(프레임)"를 그려주고, 안의 사진·이름표는 내가 채우는** 구조. 액자 디자인이 곧 카드 퀄리티.
